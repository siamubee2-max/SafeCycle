Questions fréquentes

Tout ce que vous devez savoir sur SafeCycle et la protection de vos données de santé.

🌿 Général

SafeCycle est une application de suivi du cycle menstruel conçue autour d'un principe fondamental : vos données de santé vous appartiennent, et à vous seul·e. Toutes les informations que vous saisissez (journal quotidien, douleurs, humeur, énergie, médicaments, symptômes) sont stockées localement sur votre appareil, chiffrées avec AES-256-GCM. Aucun compte, aucun serveur, aucune fuite.
SafeCycle est disponible sur iOS (iPhone, iOS 16+) et Android (Android 7.0+, API 24). Une version web progressive (PWA) est également accessible directement depuis votre navigateur.
Non. SafeCycle ne requiert aucun compte, aucune adresse email, aucun mot de passe que nous conserverions. C'est le principe du Zero-Knowledge : nous ne pouvons pas accéder à vos données car elles ne transitent jamais par nos serveurs. L'application génère localement une clé de chiffrement unique pour votre appareil.
💡 Notez qu'en cas de désinstallation ou de changement d'appareil, vos données ne pourront pas être récupérées automatiquement — vous pouvez exporter une sauvegarde JSON depuis les Paramètres.
Oui, presque entièrement. Le journal, le tableau de bord et les analyses de cycle fonctionnent 100% hors connexion. Seules les fonctionnalités IA (invite de réflexion quotidienne, analyse personnalisée) nécessitent une connexion internet pour appeler l'API Gemini. Ces prompts IA sont ensuite mis en cache localement pour la journée.
SafeCycle est entièrement traduit en 7 langues : Français, Anglais, Espagnol, Allemand, Portugais, Italien et Japonais. Vous pouvez changer de langue à tout moment depuis le sélecteur en haut de l'interface.

📋 Fonctionnalités

Le journal vous permet de consigner chaque jour :

• Douleurs : crampes abdominales (niveau 0–100) et lombalgie (on/off)
• Type de flux : spotting, léger, modéré, abondant
• Énergie : niveau personnalisable en temps réel
• Humeur : 6 émojis expressifs
• Exercice : type, durée et intensité avec suggestion adaptée à votre phase
• Médicaments : nom, dosage, heure et rappel
• Symptômes : maux de tête, ballonnements, fatigue, acné, nausées, seins sensibles, ou personnalisés
SafeCycle utilise Google Gemini 2.0 Flash pour deux fonctions :

1. Invite de réflexion quotidienne (onglet Journal) : une question personnalisée adaptée à votre phase du cycle, générée une fois par jour et mise en cache.
2. Analyse personnalisée (onglet Insights) : synthèse de vos 14 dernières entrées pour détecter des patterns (douleur, énergie, humeur) et proposer des recommandations.

Ces fonctions sont supprimées si aucune clé API n'est configurée — l'app reste entièrement utilisable sans IA.
Dans l'onglet Dashboard, la section "Focus Nutritionnel" propose un plan alimentaire de 28 jours adapté aux quatre phases du cycle (menstruelle, folliculaire, ovulatoire, lutéale). Chaque jour inclut une suggestion de recette et une description nutritionnelle. Ce plan est disponible dans les 7 langues de l'application.
Oui, SafeCycle a été pensé pour accompagner les cycles irréguliers et les pathologies associées. Depuis les Paramètres, vous pouvez sélectionner un mode de suivi spécifique : Simple, SOPK, Endométriose, ou Contraception. L'application adapte les suggestions d'exercice et les recommandations à votre profil.

Important : SafeCycle est un outil de suivi et d'information, pas un dispositif médical. Consultez toujours un professionnel de santé pour votre prise en charge.

🔒 Confidentialité & sécurité

Uniquement sur votre appareil. Vos entrées de journal, symptômes, médicaments et données de cycle sont conservées dans le stockage local de l'application (localStorage/AsyncStorage) et ne sont jamais envoyées vers un serveur externe. Aucune base de données en ligne ne contient vos données.
L'AES-256-GCM est l'un des algorithmes de chiffrement les plus robustes, utilisé par les gouvernements et institutions financières. Dans SafeCycle, une clé de chiffrement est générée localement sur votre appareil à l'activation de l'application. Ni Inference Vision, ni aucun tiers, ne peut accéder à cette clé ni déchiffrer vos données sans elle.
Jamais à des fins commerciales. Les seules transmissions de données sont :

• API Gemini (Google) : si la fonction IA est activée, un résumé anonymisé de vos données (humeur, énergie, douleur) est envoyé de manière chiffrée pour générer les insights. Aucune donnée identifiante n'est transmise.
• Expo / EAS : métriques de performance anonymes pour améliorer la stabilité de l'app.

Nous ne vendons, ne louons et ne monétisons aucune donnée personnelle.
Export : Depuis Paramètres → "Exporter mes données (JSON)", vous pouvez télécharger l'intégralité de votre journal au format JSON à tout moment.

Suppression : Depuis Paramètres → Zone dangereuse → "Supprimer toutes les données", ou simplement en désinstallant l'application (toutes les données locales sont effacées automatiquement par le système).

Comme les données sont stockées localement (et non dans le cloud), un changement d'appareil sans export préalable entraîne la perte des données.

La marche à suivre recommandée :
1. Sur l'ancien téléphone → Paramètres → Exporter mes données (JSON)
2. Conservez le fichier JSON dans votre cloud personnel (iCloud, Google Drive, etc.)
3. La future version de SafeCycle intégrera une restauration depuis ce fichier.

En cas de perte de données sans export, nous ne pouvons malheureusement pas les récupérer — c'est le corollaire du Zero-Knowledge.
L'IA nécessite une connexion internet et qu'au moins une entrée de journal ait été enregistrée pour générer une analyse. Si vous venez de commencer, enregistrez quelques jours de données puis rechargez l'onglet Insights. Si le problème persiste, vérifiez votre connexion réseau ou contactez-nous.
