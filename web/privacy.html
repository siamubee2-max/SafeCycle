Politique de Confidentialité

Dernière mise à jour : 1er avril 2026

🛡️

Principe fondateur : Zero-Knowledge

SafeCycle a été conçu avec un postulat clair : nous ne devrions jamais avoir accès à vos données de santé. Tous vos journaux, symptômes et informations médicales sont chiffrés et stockés uniquement sur votre appareil. Nous ne sommes pas en mesure de les lire, les vendre, ni les partager — physiquement impossible.

1. Introduction

SafeCycle (« l'Application ») est éditée par Inference Vision, studio indépendant fondé par Christophe G. Weber. Cette politique de confidentialité décrit les pratiques de traitement des données liées à l'utilisation de l'Application sur iOS, Android et sur le web.

En utilisant SafeCycle, vous acceptez les pratiques décrites ci-dessous. Si vous n'êtes pas d'accord, veuillez ne pas utiliser l'Application.

Cette politique s'applique à toutes les versions de SafeCycle (mobile et web progressive).

2. Données que nous traitons

2.1 Données stockées localement (sur votre appareil)

L'intégralité des données suivantes est stockée exclusivement sur votre appareil, chiffrée avec AES-256-GCM. SafeCycle n'y a jamais accès :

  • Journal quotidien : niveau de crampes, lombalgie, type de flux, niveau d'énergie, humeur
  • Exercice : type, durée, intensité
  • Médicaments : nom, dosage, heure de prise, rappels
  • Symptômes : liste, intensité, notes personnelles
  • Prompts IA : invites de réflexion quotidiennes mises en cache localement
  • Préférences : langue, mode sombre, mode de suivi (Simple / SOPK / Endométriose / Contraception)

2.2 Données transmises à des services tiers (optionnel)

Uniquement si la fonctionnalité IA est activée, un résumé anonymisé de vos 14 dernières entrées (humeur, énergie, douleur, flux — sans aucune donnée identifiante) est envoyé chiffré (TLS 1.3) à l'API Google Gemini pour générer des insights personnalisés. Ce résumé ne contient ni nom, ni email, ni identifiant quelconque.

2.3 Données collectées automatiquement

  • Métriques de performance anonymes via Expo / EAS (crashs, temps de chargement) — aucune donnée de santé n'est incluse.
❌ Données que nous ne collectons absolument pas
  • Nom, prénom, adresse email (sauf si vous nous contactez)
  • Numéro de téléphone
  • Données de paiement (gérées exclusivement par Apple / Google)
  • Contacts, photos personnelles, galerie photo
  • Historique de navigation
  • Identifiants publicitaires (IDFA / GAID)
  • Localisation géographique

3. Finalités du traitement

Les données décrites au point 2.2 (résumé anonymisé envoyé à l'IA) sont utilisées exclusivement pour :

  • Générer des invites de réflexion personnalisées adaptées à votre phase du cycle
  • Produire une analyse santé synthétique et des recommandations de bien-être
  • Améliorer la stabilité technique de l'Application (métriques Expo anonymes)

Nous n'utilisons jamais vos données à des fins publicitaires, de profilage commercial, ou de vente de données.

4. Services tiers

  • Google Gemini API (ai.google.dev) — Analyse IA optionnelle. Soumis à la politique de confidentialité Google. Aucune donnée identifiante n'est transmise.
  • Expo / EAS (expo.dev) — Infrastructure de build et distribution. Collecte des métriques de performance anonymes. Soumis à la politique de confidentialité d'Expo.
  • Apple App Store / Google Play Store — Gèrent les achats in-app (SafeCycle Lifetime). Soumis à leurs propres politiques de confidentialité.
  • Google Fonts — Chargement de polices de caractères (Noto Serif, Manrope). Peut collecter l'adresse IP de votre appareil selon la politique Google.

5. Sécurité des données

Chiffrement local AES-256-GCM : une clé de chiffrement unique est générée sur votre appareil lors de la première ouverture. Ni Inference Vision ni aucun tiers ne peuvent déchiffrer vos données sans cette clé.

  • Toutes les données locales sont stockées dans le sandbox sécurisé de l'application
  • Les transmissions réseau utilisent TLS 1.3 (HTTPS)
  • Aucune base de données externe ne contient vos données personnelles de santé
  • Aucun cookie de tracking n'est utilisé dans l'Application

La clé de chiffrement locale n'est pas sauvegardée par Inference Vision. En cas de perte (désinstallation, réinitialisation d'appareil), les données ne sont pas récupérables — nous recommandons d'exporter régulièrement votre journal en JSON via les Paramètres.

6. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (UE) 2016/679, vous disposez des droits suivants :

  • Droit d'accès : vous pouvez consulter toutes vos données via la fonction "Exporter mes données" dans les Paramètres de l'Application.
  • Droit de rectification : vos données peuvent être modifiées directement dans l'Application à tout moment.
  • Droit à l'effacement : via Paramètres → Zone dangereuse → Supprimer toutes les données, ou en désinstallant l'Application.
  • Droit à la portabilité : export JSON disponible dans les Paramètres.
  • Droit d'opposition : vous pouvez désactiver la fonctionnalité IA dans les Paramètres pour qu'aucune donnée ne soit jamais transmise à des tiers.

Pour exercer ces droits ou pour toute question : inferencevision@inferencevision.store

7. Conservation des données

Les données stockées localement persistent tant que l'application est installée sur votre appareil. La désinstallation de SafeCycle supprime automatiquement l'intégralité des données locales (journal, médicaments, symptômes, préférences, cache IA).

Les métriques de performance anonymes collectées par Expo sont conservées selon leur propre politique (généralement 90 jours).

8. Utilisateurs mineurs

SafeCycle s'adresse aux personnes de 16 ans et plus. Nous ne collectons pas sciemment de données personnelles concernant des enfants de moins de 16 ans. Si vous êtes parent ou tuteur et pensez que votre enfant a utilisé l'Application, contactez-nous à inferencevision@inferencevision.store pour suppression immédiate des données éventuelles.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. En cas de modification substantielle affectant vos droits, une notification sera affichée à l'ouverture de l'Application. La date de dernière mise à jour est toujours indiquée en haut de ce document.

L'utilisation continue de SafeCycle après publication d'une mise à jour vaut acceptation de la nouvelle politique.
